# vgm

> A web application for Video Game Music (VGM) management and playback.

## Description

VGM is a web application for organizing and playing Video Game Music tracks. Built with .NET and featuring PowerShell publish scripts, it provides a web interface for browsing and streaming VGM audio files with Azure/Firebase backend integration.

## Stack / Tech
- .NET / ASP.NET MVC
- PowerShell (deployment scripts)
- Azure / Firebase

## Getting Started

```bash
dotnet run --project VGM.Web/VGM.Web.csproj
```
